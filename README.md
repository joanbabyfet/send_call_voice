## About
通过 twilio 服务实现短信和电话提醒功能，提供推广人员根据名单批量通知不同客户

## Feature

* 通过 twilio 服务, 引用 twilio 库, 实现短信或电话提醒
* 根据给定的名单列表 list.csv 进行发送
* 按日期单独生成日志文件
* 配置文件设置SID和Token

## Requires
Python 3.11.0  
twilio 7.16.1  
configparser 5.3.0  
pandas 1.5.2  

## Usage
```
python main.py
```
